# Weibo
模仿新浪微博, 采用 Swift 语言

## 实现功能

* 首页数据的加载显示, 以及下拉刷新和上拉加载更多数据
* 点击查看大图的自定义弹出和消失动画
* 保存图片到相册
* 发布微博页面自定义表情键盘

其他页面没有实现, 主要是模仿学习首页复杂数据的加载显示, 图片浏览, 表情键盘等功能
